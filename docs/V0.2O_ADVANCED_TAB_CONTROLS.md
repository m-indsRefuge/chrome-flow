# V0.2O — Advanced Deterministic Tab Controls

## Purpose

V0.2O adds the remaining deterministic tab-management features that are useful before Chrome Flow moves into UI design and AI command-contract work.

This slice keeps the phase-complete layer approach:

```text
1. Complete deterministic tab management.
2. Validate the behavior locally.
3. Then discuss UI redesign safely.
4. Then draft the AI command contract against stable tools.
```

## Architecture Note

The V0.2N controller remains the native tab-management base.

V0.2O adds a separate advanced control surface:

```text
src/sidepanel/advanced-tab-controls.js
src/sidepanel/advanced-tab-controls.css
```

This is not an override module. It does not intercept or replace existing sidepanel controller behavior. It adds new deterministic actions in the same style as Workspace Session Control.

## Added Features

### Chrome group collapse / expand

Controls:

- Collapse Workspace Chrome Groups
- Expand Workspace Chrome Groups

These operate only on native Chrome tab groups that contain currently resolved workspace tabs.

### Move workspace tabs into one window

Control:

- Move Workspace Tabs Into One Window

This brings all open resolved workspace tabs into one Chrome window without closing tabs.

### Arrange tabs by role order

Control:

- Arrange Tabs by Role Order

This orders open workspace tabs inside each window according to the current workspace type's role order.

### Pin / unpin workspace tabs

Controls:

- Pin Workspace Tabs
- Unpin Workspace Tabs

These explicitly pin or unpin all open resolved workspace tabs.

### Reopen all missing tabs

Control:

- Reopen All Missing Tabs

This reopens safely missing workspace tabs. Ambiguous records are skipped rather than guessed.

### Export URL list

Control:

- Copy Workspace URL List

This copies a Markdown URL list grouped by workspace role.

### Duplicate URL review

Surface:

- Duplicate URL Review

The review surface groups same-URL workspace records together and allows the Operator to:

- keep records by doing nothing
- apply aliases
- focus a duplicate record
- remove a duplicate workspace record without closing the browser tab

## Deferred Features

The following remain deferred:

- drag-and-drop tab reorder
- saved tab templates
- automatic tab classification
- AI recommendations for closing/removing tabs
- semantic tab clustering

These belong to later UI, workspace-management, or AI-layer sprints.

## AI Contract Relevance

V0.2O adds deterministic actions that future AI can propose through a tool contract, such as:

- collapse workspace groups
- expand workspace groups
- move workspace tabs into one window
- arrange tabs by role
- pin/unpin workspace tabs
- reopen missing tabs
- export URL list
- review duplicate URLs

The AI layer should propose these actions; it should not directly manipulate Chrome APIs.

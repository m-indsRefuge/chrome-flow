# Tests

Initial manual test checklist:

1. Load extension in Chrome Developer Mode.
2. Open side panel.
3. Click Load Current Window Tabs.
4. Confirm tabs appear.
5. Add workspace name.
6. Add workspace aim.
7. Add tab aliases.
8. Add tab roles.
9. Add journal entry.
10. Close and reopen side panel.
11. Confirm workspace state persists.

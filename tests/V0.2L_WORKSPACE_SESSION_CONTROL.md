# V0.2L Workspace Session Control Checklist

## UI validation

1. Pull latest repo and reload Chrome Flow.
2. Confirm Workspace Session Control appears below the Workspace section.
3. Confirm the summary shows active workspace name, tab count, user note count, system event count, and archived workspace count.
4. Confirm buttons appear:
   - Archive Current Workspace
   - Archive + Start Fresh Workspace
   - Copy Workspace Snapshot

## Copy snapshot validation

1. Click Copy Workspace Snapshot.
2. Paste the clipboard contents into a temporary text file.
3. Confirm it is JSON.
4. Confirm it includes packetType: Chrome Flow Workspace Snapshot.
5. Confirm it includes activeWorkspace.
6. Confirm it includes archiveSummary.
7. Confirm Developer Diagnostics records workspace_snapshot_copied.

## Archive current validation

1. Click Archive Current Workspace.
2. Confirm the browser asks for confirmation.
3. Confirm the active workspace remains visible after archiving.
4. Confirm the archived workspace count increases by one.
5. Confirm Developer Diagnostics records workspace_archived.

## Archive and start fresh validation

1. Click Archive + Start Fresh Workspace.
2. Confirm the browser asks for confirmation.
3. Confirm the side panel reloads.
4. Confirm the new active workspace is Untitled Workspace.
5. Confirm workspace tabs are empty.
6. Confirm User Journal is empty.
7. Confirm System Journal is empty.
8. Confirm Recovery Journal is empty.
9. Confirm browser tabs remain open.
10. Confirm Developer Diagnostics remains available.

## Safety validation

1. Confirm Clear Diagnostics still only clears diagnostics.
2. Confirm Workspace Session Control does not close browser tabs.
3. Confirm starting fresh does not erase the archive summary count.

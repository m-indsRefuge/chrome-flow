# V0.2K Tab Intake Instance Matching Checklist

## Duplicate URL validation

1. Pull latest repo and reload Chrome Flow.
2. Clear workspace tabs.
3. Open two different Chrome tabs to the same URL.
4. Open at least one third tab to a different URL.
5. Click Scan Current Window Tabs.
6. Select both same-URL tabs and the third tab.
7. Click Add Selected Tabs to Workspace.
8. Confirm all three selected tab instances are added when none are exact existing workspace tab instances.
9. Confirm the workspace tab count matches the number of added tab instances.
10. Click Scan Current Window Tabs again.
11. Confirm exact existing workspace tab instances are disabled and show Already in workspace.
12. Open another new tab to the same URL as one workspace tab.
13. Scan again.
14. Confirm the new same-URL tab remains selectable.
15. Confirm the new same-URL tab shows Same URL already in workspace.
16. Add the same-URL duplicate tab.
17. Confirm Chrome Flow adds it as a separate workspace tab instance.

## Diagnostics validation

1. Click Copy Diagnostic Packet.
2. Confirm the recent system event includes selected_tabs_added.
3. Confirm the selected_tabs_added event includes intakeMatchingMode: instance_aware.
4. Confirm result counts include addedCount, exactSkippedCount, duplicateUrlAddedCount, and missingCount.
5. Confirm no runtime error is reported.

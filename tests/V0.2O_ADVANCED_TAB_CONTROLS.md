# V0.2O Advanced Deterministic Tab Controls Checklist

## Load validation

1. Pull latest repo and reload Chrome Flow.
2. Confirm the side panel loads without runtime errors.
3. Confirm Advanced Tab Controls appears below Workspace Tabs.
4. Confirm the following controls appear:
   - Collapse Workspace Chrome Groups
   - Expand Workspace Chrome Groups
   - Move Workspace Tabs Into One Window
   - Arrange Tabs by Role Order
   - Pin Workspace Tabs
   - Unpin Workspace Tabs
   - Reopen All Missing Tabs
   - Copy Workspace URL List
   - Refresh Duplicate URL Review
5. Confirm Duplicate URL Review appears.

## Group collapse / expand validation

1. Add several tabs to the workspace.
2. Assign roles.
3. Click Create Chrome Tab Groups.
4. Click Collapse Workspace Chrome Groups.
5. Confirm workspace Chrome groups collapse.
6. Click Expand Workspace Chrome Groups.
7. Confirm workspace Chrome groups expand.
8. Confirm System Journal records chrome_tab_groups_collapsed and chrome_tab_groups_expanded.

## Move into one window validation

1. Put workspace tabs across two Chrome windows.
2. Click Move Workspace Tabs Into One Window.
3. Confirm all open workspace tabs move into one Chrome window.
4. Confirm no tabs close.
5. Confirm System Journal records workspace_tabs_moved_to_one_window.

## Arrange by role validation

1. Assign several workspace tabs to different roles.
2. Click Arrange Tabs by Role Order.
3. Confirm tabs are ordered by the role order for the selected workspace type.
4. Confirm System Journal records workspace_tabs_arranged_by_role.

## Pin / unpin validation

1. Click Pin Workspace Tabs.
2. Confirm all open workspace tabs are pinned.
3. Click Unpin Workspace Tabs.
4. Confirm all open workspace tabs are unpinned.
5. Confirm System Journal records workspace_tabs_pinned and workspace_tabs_unpinned.

## Reopen missing validation

1. Add several tabs to the workspace.
2. Close one browser tab outside Chrome Flow.
3. Click Refresh Workspace Tab Metadata.
4. Confirm the closed tab appears as missing.
5. Click Reopen All Missing Tabs.
6. Confirm the missing tab reopens.
7. Confirm ambiguous tabs are skipped rather than guessed.
8. Confirm System Journal records missing_workspace_tabs_reopened.

## URL export validation

1. Add workspace tabs across several roles.
2. Click Copy Workspace URL List.
3. Paste into a temporary text file.
4. Confirm the output is Markdown.
5. Confirm URLs are grouped by role.
6. Confirm System Journal records workspace_url_list_copied.

## Duplicate URL review validation

1. Add two separate browser tabs with the same URL to the workspace.
2. Click Refresh Duplicate URL Review.
3. Confirm the duplicate URL group appears.
4. Apply an alias to one duplicate.
5. Confirm System Journal records duplicate_tab_alias_updated.
6. Click Focus Tab for one duplicate and confirm the correct browser tab is focused.
7. Remove one duplicate record.
8. Confirm the browser tab remains open.
9. Confirm Recovery Journal includes a restore-capable workspace_tab_removed event.

## Packet validation

1. Copy Active Workspace Packet.
2. Copy Diagnostic Packet.
3. Confirm no runtime errors.
4. Confirm pendingActionTraces is empty after all actions complete.
5. Paste both packets into the build thread for review.

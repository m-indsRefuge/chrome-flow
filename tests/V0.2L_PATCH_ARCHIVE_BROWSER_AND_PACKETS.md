# V0.2L Patch — Archive Browser and Packet Checklist

## Archive browser validation

1. Pull latest repo and reload Chrome Flow.
2. Confirm Workspace Session Control appears below the Workspace section.
3. Confirm Archived Workspaces selector appears.
4. Confirm existing archive entries appear in the selector.
5. Select an archived workspace.
6. Confirm selected archive summary shows:
   - Archive name
   - Reason
   - Workspace name
   - Workspace type
   - Tabs
   - User notes
   - System events

## Active workspace packet validation

1. Click Copy Active Workspace Packet.
2. Paste into a temporary text file.
3. Confirm it is JSON.
4. Confirm packetType is Chrome Flow Workspace Packet.
5. Confirm extension.schema is workspace-packet-v0.2.
6. Confirm source.type is active_workspace.
7. Confirm activeWorkspace is included.
8. Confirm archiveSummary is included.
9. Confirm diagnostics records workspace_packet_copied.

## Selected archived workspace packet validation

1. Select an archived workspace from the selector.
2. Click Copy Selected Archive Packet.
3. Paste into a temporary text file.
4. Confirm it is JSON.
5. Confirm packetType is Chrome Flow Archived Workspace Packet.
6. Confirm extension.schema is archived-workspace-packet-v0.1.
7. Confirm source.type is archived_workspace.
8. Confirm selectedArchive.workspace is included.
9. Confirm archiveSummary is included.
10. Confirm diagnostics records archive_packet_copied.

## Safety validation

1. Confirm copying an archive packet does not change the active workspace.
2. Confirm selecting an archive does not restore it automatically.
3. Confirm no restore action exists yet.
